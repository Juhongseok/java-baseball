1. 랜덤 숫자 생성
    - 서로 다른 수
    - 3자리 자연수
2. 사용자 입력
    - 입력값 검증
        - 3자리 서로 다른 자연수
        - 통과 할 경우 다음 단계로
        - 통과하지 못할 경우 IllegalArgumentException 발생, 종료
3. 입력값 정답 비교
    - 스트라이크: 같은 숫자, 같은 위치
    - 볼: 같은 숫자, 다른 위치
    - 낫싱: 전부 다른 숫자
    - 모두 스트라이크일 경우 성공, 다음 단계로
4. 재시작/종료 코드값 입력
    - 성공시 실행
    - code 1 재시작
    - code 2 종료
